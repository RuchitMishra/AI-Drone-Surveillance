# AI Drone Surveillance System

This repository contains my ongoing research and development work related to AI enabled drone surveillance systems using Computer Vision and deep learning techniques.

## Project Objective

The primary goal of this project is to explore the use of autonomous UAV systems for intelligent aerial surveillance and monitoring applications especially for border surveillance scenarios.

## Current Work

* Exploring object detection models such as YOLO
* Studying aerial datasets including VisDrone
* Training Computer Vision models for aerial object detection
* Experimenting with drone imagery preprocessing and annotations
* Understanding deployment challenges in UAV based surveillance systems

## Future Goals

* Real time aerial object detection
* Autonomous drone monitoring pipeline
* Multi drone coordination systems
* AI assisted border surveillance
* Drone communication optimization and secure UAV networking

## Research Interests

Along with surveillance systems I am also interested in drone communication technologies and autonomous UAV coordination systems. I have been studying research papers related to UAV networking and protocols including Chimera protocol for improved drone communications.
